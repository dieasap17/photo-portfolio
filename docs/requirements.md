# 需求文档 — 摄影作品集网站

## 项目概述

个人摄影/视频作品集网站，面向面试官展示摄影与视频创作能力。单页面结构，包含作品展示区与个人信息区。

## 核心需求

### 内容展示
- 展示摄影视频作品，支持 4K 和 1080p 分辨率
- 按类别分类：人物、产品、剧情（类别可增减）
- 每个类别可包含多个视频片段（数秒至数十秒）
- 每页展示 4 个视频卡片，超出时翻页浏览
- 卡片默认显示封面图（支持上传视频时自选帧作为封面），点击后播放完整视频

### 页面结构
- 单页面设计：作品展示区 + 关于我区
- 顶部导航：姓名标识 + 作品锚点
- 作品区：居中标题 + 分类标签 + 视频卡片网格 + 翻页
- 关于我区：头像 + 姓名 + 个人简介 + 工作经历 + 联系方式

### 设计风格
- 留白极简风格（MUJI 感）
- 配色：Morandi 低饱和色系 + 托斯卡纳暖调 + 鼠尾草绿互补点缀
- 亚麻布纹纹理背景
- 圆角卡片 + 细边框
- 粗线条分隔设计
- 参考数码科技大厂官网质感

### 视频处理
- 方案 A（主）：轻度 H.265 压缩后直接部署，优化文件大小同时保留画质
- 方案 C（备选）：阿里云 OSS / 七牛云 对象存储 + CDN 加速

### 部署要求
- 免费托管平台
- **国内可直接访问**
- 生成链接即可分享

### 动效
- 适当的页面动画效果，提升质感
- 不喧宾夺主，服务于内容展示

## 用户画像
- 摄影爱好者，不懂代码
- 需要能自行更新作品内容（后续可扩展 CMS）
- 当前场景：分享链接给面试官查看作品集
