# 开发计划 — 摄影作品集网站

## 开发原则

- **小步推进**：每个阶段独立可验证，完成后确认再进入下一阶段
- **先骨架后血肉**：先搭结构 → 再加内容 → 最后打磨
- **每阶段结束**：更新 DEVLOG.md，标记完成事项

---
