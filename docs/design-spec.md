# 设计规范 — 摄影作品集网站

## 设计方向

**留白极简 × Morandi × 托斯卡纳** — 大面积暖白底搭配低饱和大地色系，鼠尾草绿为互补点缀，亚麻纹理增加触感，克制而温暖。

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 底色 | `#faf9f7` | 暖白，页面背景 |
| 浅色块 | `#c8bba0` ~ `#e0d5be` | 卡片/占位底色，各卡片微调区分 |
| 辅助文字 | `#8c7b65` / `#6b5d4e` | 非激活分类、二级信息 |
| 主文字 | `#2d2416` | 标题、激活态、重点信息 |
| 分割线 | `#d4cdb8` | 横向粗线条 |
| 强调线 | `#8b7860` | 选中态下划线、重点分割 |
| 互补点缀 | `#8a9a7b` | 鼠尾草绿 — 时长角标、小圆点、渐变尾端 |
| 标签底色 | `#edebe4` | 浅灰米色标签背景 |
| 封面播放按钮 | `rgba(45,36,22,0.65)` | 半透明深褐圆形底 + 浅米三角 |

## 组件规范

### 视频卡片
- 宽高比：4:3
- 圆角：8px
- 边框：1.5px solid（颜色依卡片微调）
- 默认显示：封面图 + 居中圆形播放按钮
- 悬浮态：播放按钮微微放大（~5%）
- 点击后：卡片内播放完整视频
- 时长角标：右下角，半透明深色底

### 分类标签
- 水平排列，间距 40px
- 激活态：文字色 `#2d2416` + 底部 3px 粗线 `#8b7860`
- 非激活态：文字色 `#b8a899` + 底部 1px 细线 `#d4cdb8`
- 悬停：文字色渐变为 `#6b5d4e`

### 翻页控件
- 居中排列
- 上一页 / 圆点组 / 下一页
- 当前页圆点：`#8b7860`
- 非当前页圆点：`#d4cdb8`

### 关于我信息字段
- 字段标签大写 + 宽字间距
- 字段下方细线分隔
- 内容区斜体占位

## 纹理

背景叠加亚麻纹理：
```css
background-image: repeating-linear-gradient(
  0deg,
  transparent,
  transparent 2px,
  rgba(170, 155, 135, 0.07) 2px,
  rgba(170, 155, 135, 0.07) 4px
);
```

## 响应式设计

### 断点
| 断点 | 宽度 | 布局 |
|------|------|------|
| 桌面 | ≥ 768px | 4 列卡片 + 完整间距 |
| 手机 | < 768px | 2 列卡片 + 紧凑间距 |

### 手机端适配规则
- 卡片网格：4 列 → 2 列
- 间距减半（gap: 16px → 8px）
- 字号等比缩小约 20%
- 分类标签间距缩小（40px → 12px）
- 关于我区域：头像 + 姓名右对齐，简历信息居中
- 翻页控件保持居中

### 视频放大播放（手机端）
- 点击卡片 → 视频进入全屏播放模式（黑色背景）
- 底部显示：播放/暂停 + 进度条 + 时间 + 全屏/关闭按钮
- 下滑或点关闭按钮 → 返回浏览模式
- 桌面端可选同样交互，或在卡片内直接播放
